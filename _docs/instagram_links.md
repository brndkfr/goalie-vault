## Snippet

```javascript
var found = new Set();
var obs = new MutationObserver(() => {
  document.querySelectorAll('a[href*="/p/"]').forEach(a => found.add(a.href));
});
obs.observe(document.body, { childList: true, subtree: true });
document.querySelectorAll('a[href*="/p/"]').forEach(a => found.add(a.href));
console.log(`Watching... ${found.size} found. Scroll down slowly now.`);
window.stopCollecting = () => {
  obs.disconnect();
  var blob = new Blob([[...found].join('\n')], {type:'text/plain'});
  var a = document.createElement('a'); a.href = URL.createObjectURL(blob);
  a.download = 'instagram_links.txt'; document.body.appendChild(a); a.click(); a.remove();
  console.log(`Done! ${found.size} links saved to file.`);
};
```

### How to use

1. Open the Instagram saved collection page in Firefox
2. Press **F12** → **Console** tab
3. Scroll to the **top** of the collection
4. Paste the snippet above and press **Enter**
5. **Scroll down slowly** through all posts — watch the console count increase
6. When you reach the bottom, type **`stopCollecting()`** and press **Enter**
7. A file `instagram_links.txt` will download with all the links
